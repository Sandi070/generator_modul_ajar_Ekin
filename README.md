Aplikasi web sumber terbuka (*open-source*) yang dirancang untuk membantu rekan-rekan pendidik dalam menyusun **Modul Ajar (RPP)** secara otomatis, terstruktur, dan komprehensif. Aplikasi ini ditenagai oleh **Google Gemini AI** dan dibangun menggunakan **Streamlit**.

Keunggulan utama generator ini adalah kemampuannya untuk mengintegrasikan **Pendekatan Pembelajaran Mendalam (Deep Learning)** dan **Kurikulum Berbasis Cinta** khas Direktorat Jenderal Pendidikan Islam (Pendis) Kementerian Agama RI, yang mengedepankan empati, kasih sayang, dan pendekatan humanis yang berpusat pada siswa.

## ✨ Fitur Utama
* **Kustomisasi Fleksibel:** Mendukung berbagai mata pelajaran, fase, dan kelas.
* **Struktur Modul Lengkap:** Menghasilkan Informasi Umum, Komponen Inti (termasuk asesmen diagnostik, formatif, sumatif beserta rubrik), Pengayaan/Remedial, hingga Lampiran (Bahan Ajar & LKPD).
* **Kurikulum Berbasis Cinta:** Otomatis merangkai kalimat apersepsi dan penutup yang bernuansa kasih sayang dan menghargai proses belajar anak.
* **Pendekatan Tambahan Spesifik:** Guru dapat memilih untuk memfokuskan modul pada:
    * Pendidikan Karakter
    * Moderasi Beragama (Nilai-nilai Kemenag)
    * Pembelajaran Bilingual (Indonesia - Inggris)
    * Berbasis TIK (Integrasi simulasi PhET, AR/VR, dll)
    * Pembelajaran Inovatif (PBL/PjBL dengan Gamifikasi)
* **Privasi Terjaga:** Menggunakan sistem *input* API Key mandiri, sehingga setiap guru menggunakan kuota AI mereka masing-masing dengan aman.
